# ag-grid-bootswatch

Variations on the ag-grid theme theme-bootstrap for all the themes from the bootswatch project.

## Features

- Based on bootstrap 4 and bootswatch 4 

## Development

```
yarn install
npm run build
```

## Use in your project

```
yarn add ag-grid-bootswatch 
```

Or

```
npm install ag-grid-bootswatch --save 
```
